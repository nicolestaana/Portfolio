# Personal Portfolio

This is my official professional portfolio website, designed to showcase my journey as a **Computer Engineering** student, my skills, and my academic achievements at the **Polytechnic University of the Philippines**.

## Live Preview
You can view the live site here: 
[https://nicolestaana.github.io/portfolio/](https://nicolestaana.github.io/Portfolio/) 

## Features
* **Responsive Navigation:** Smooth scrolling to About, Skills, Education, and Contact sections.
* **Academic Timeline:** A detailed look at my education.
* **Skill Matrix:** Organized display of technical languages (Python, HTML, CSS, JS) and tools.
* **Quick Contact:** Integrated social icons and a direct "Download Resume" feature.

## File Structure
* `index.html` - The core structure of the portfolio.
* `style1.css` - Custom styles for layout, typography, and buttons.
* `picture.png` - Formal profile image.
* `STA. ANA, Nicole Decemae M_Resume.pdf` - Downloadable professional resume.
* `/icons` - Directory for social and platform icons.

## About Me
I am a detail-oriented Computer Engineering student passionate about learning new tools and environments. My goal is to apply my problem-solving skills to real-world technical challenges through collaboration and continuous innovation.
